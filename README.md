# tfm_pH_prediccion_analisis_tendencia
Este trabajo aborda la tarea de reconstruir series temporales de pH en el mar Balear y estimar las tasas de acidificación oceánica por masa de agua en la región mediante el uso de técnicas de aprendizaje automático, contribuyendo a reducir la incertidumbre sobre los efectos del cambio climático en el Mediterráneo occidental.
